CMSimple_XHは主にPHPで開発されています。 UX（ユーザーエクスペリエンス）を向上させるには、ブラウザ側のスクリプトが必要です。 これはブラウザのECMAScript実装で直接行われます（つまり、CoffeeScriptやTypeScriptなどのプリコンパイルされた言語は使用されません）。
これは別のJavaScriptドキュメントに記載されています 。

jQuery

jQuery はCMSimple_XHによって正式に承認された唯一のJavaScriptフレームワークです。
すべてのCMSimple_XH拡張ライターは、jQuery、jQueryUIおよび/またはjQueryプラグインを使用する場合は、 jQuery4CMSimple を使用することをお勧めします。
　※いくつかの主要なJavaScriptフレームワークがお互いに互換性にまだ不安があります。
　※jQuery4CMSimple　は、コアに　plugins/jquery　として組み込まれてます。
　　　コンフリクトや多重読み込みを防止する　単純なEnqueue System　が備わってます
